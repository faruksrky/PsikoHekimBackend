# Arşiv - Eski Docker Compose Dosyaları

Eski monolitik compose yapısından kalan dosyalar.
Yeni yapı: 1-keycloak, 2-psikohekim, 3-bpmn (her proje ayrı)

- docker-compose.prod.yml - Tüm servisler (postgres, redis, keycloak, backend)
- docker-compose.yml - Profil tabanlı (bpmn, backend)
- docker-compose.dev.yml - Dev ortamı
- docker-compose.keycloak.yml - Eski Keycloak+Postgres
- docker-compose.keycloak-fresh.yml - Boş Keycloak kurulumu
